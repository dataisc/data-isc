# Sample Datasets

Drag any of these files onto the **Upload your dataset** button on the main page.

| File | What to explore |
|------|-----------------|
| `co2_emissions.csv` | Project CO₂ trajectories, apply a "green policy" scenario |
| `life_expectancy.csv` | Compare health convergence, apply a "pandemic shock" scenario |
| `renewable_energy_share.csv` | Energy transition trends, apply a "rapid decarbonisation" scenario |

All data is sourced from publicly available datasets (World Bank, Our World in Data, IEA) and is illustrative — not a complete dataset.
