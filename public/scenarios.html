Scenarios

Built-in what-if scenarios for every demo dataset — assumptions, mechanics, and interpretation

Data ISC ships three ready-to-run scenarios for each demo dataset, calibrated to realistic real-world mechanisms. Load any demo dataset from the home screen and click a scenario card to apply it instantly. The engine projects the baseline forward and then applies the scenario delta year-by-year so you can compare trajectories side-by-side.

Black Swan Library — Any Dataset

Black Swan presets are available for every dataset — demo and uploaded alike. A 🦢 Browse Black Swan presets button appears below the scenario cards in the What-If panel. Each preset applies a dispersed delta across entities, calibrated to the relative size, momentum, or volatility of each entity in your dataset. They are deliberately broad — use them as a first pass and refine with per-entity overrides or AI analysis.

How scenarios work

All scenarios apply a percentage growth rate delta to the projected value of each affected entity, year by year, over the defined window. Some represent sustained structural changes; others phase out gradually. The chart overlays the scenario trajectory as a dashed line on top of the baseline, with a shaded impact band — green for beneficial changes, red for detrimental ones (direction is automatically detected from the metric type). See Methodology for the full mechanics.

Running multiple scenarios

The simulator supports stacking multiple scenarios simultaneously. Deltas are additive — there is no interaction term. This means stacking a +1 pp trade liberalisation with a −0.5 pp carbon tax produces a net +0.5 pp effect for entities affected by both. Non-linear interactions are not modelled.

Comparing two branches

The Trajectories chart includes a ⑂ Branch mode for comparing two futures head-to-head. Pick a branch year, then assign each branch its own stack of scenarios — for example Branch A: Green Hydrogen Scaling versus Branch B: Fossil Fuel Rebound. Both timelines are computed independently and drawn on the same graph, splitting from the branch year onward. A shaded band fills the gap between them, and delta cards report the divergence at the final year and where the gap is widest.

Proposing new scenarios

Dataset-specific scenario definitions live in the DATASET_PRESETS object in public/index.html; GDP scenarios are in models/gdp-trajectories/data/scenarios_policy.json. To propose a new scenario, open a pull request with the definition and a short explanation of the effect parameters. See the GitHub repository for the schema.
